Configuração · Regex
Padrão da Referência
Regex para capturar a referência. Use grupo 1 (...) para o valor.
Ex: P([A-Z0-9]+) captura após "P"
Padrão da Quantidade
Regex para capturar a quantidade. Use grupo 1 para o número.
Ex: Q\d{3}(\d{3})\d{3} captura os 3 dígitos do meio de Q000010000
Aplicar transformação na quantidade (dividir por 10, 100...)
Configuração · Pipe
Formato: campo1|campo2|campo3...
Defina em qual posição (índice 0 = primeiro) está cada informação.
Configuração · Chave-Valor
Formato: CHAVE=valor;CHAVE2=valor2
Informe os nomes das chaves usadas neste fornecedor.
Configuração · Posicional
Extrai valores por posição fixa no texto do QR code.
Posições começam em 0. Início:Fim (fim exclusivo).

Padrão SPI · OCR
Ativado automaticamente ao digitar SPI
Referência: I + 11 dígitos + 1 letra
Quantidade: campo Qtd seguido do número

Exemplo de etiqueta:
I12345678901A
Qtd: 240
Resultado: Ref I12345678901A · Qtd 240
Padrão LINKPLAS
Ativado automaticamente ao digitar LINKPLAS
Referência: segmento P{REF} até o próximo espaço
Quantidade: Q000{QTY}000 — 3 dígitos centrais

Exemplo:
S049... P4901974V000 Q000010000 1T...
Resultado: Ref 4901974V000 · Qtd 10
